# 2048 Game

A classic 2048 puzzle game built with vanilla JavaScript, HTML, and CSS.
Slide tiles in four directions, merge numbers, and try to reach the 2048 tile!
- [Play the live version](https://sborichevskyi.github.io/2048-game/)

## Features

- Merge and spawn effects for improved gameplay
- Keyboard control support (Arrow keys / WASD)
- Score tracking
- Responsive layout
- Clean and modular codebase

## Technologies used:
- HTML5
- CSS3
- JavaScript (ES6+)
- DOM manipulation
- Game state management

## Local Setup
### Follow these steps to run the project locally:
1. Clone the repository
`git clone https://github.com/sborichevskyi/2048-game.git`
`cd 2048-game`

2. Install dependencies
`npm install`
or
`yarn install`

3. Start the project locally
`npm start`
or
`yarn start`
